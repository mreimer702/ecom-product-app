# In-N-Out Menu App

## Overview
This is a React-based application that displays the In-N-Out Burger menu. Users can toggle between the regular and secret menu items and click on individual items to view additional details such as description, price, and an image.

## Features
- **Toggle between regular and secret menu**
- **Click on menu items to reveal more details**
- **Displays item name, image, description, and price**
- **Interactive UI with React state management**

## Technologies Used
- React.js
- JavaScript (ES6+)
- CSS for styling
- PropTypes for type checking

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/in-n-out-menu.git
   ```
2. Navigate to the project directory:
   ```bash
   cd in-n-out-menu
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm start
   ```

## Usage
1. Click on **'Show Secret Menu'** to toggle between regular and secret items.
2. Click on any **menu item** to reveal its description, price, and image.
3. Click again on the item to hide the details.
