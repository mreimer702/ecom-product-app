In-N-Out Menu App

Overview

This is a React-based application that displays the In-N-Out Burger menu. Users can toggle between the regular and secret menu items and click on individual items to view additional details such as description, price, and an image.

Features

Toggle between regular and secret menu

Click on menu items to reveal more details

Displays item name, image, description, and price

Interactive UI with React state management

Technologies Used

React.js

JavaScript 

CSS for styling

Installation

Clone the repository:

git clone https://github.com/your-username/in-n-out-menu.git

Navigate to the project directory:

cd in-n-out-menu

Install dependencies:

npm install

Start the development server:

npm start

Usage

Click on 'Show Secret Menu' to toggle between regular and secret items.

Click on any menu item to reveal its description, price, and image.

Click again on the item to hide the details.
